# Exirující blog

## Jak spustit (dev)

1 Instalace balíčků

```bash
pip install -r requirements.txt && python manage.py tailwind install
```

2 Inicializace databáze
```bash
python manage.py manage.py migrate
```

3.2 Spuštění vývojového serveru

```bash
python manage.py runserver
```
